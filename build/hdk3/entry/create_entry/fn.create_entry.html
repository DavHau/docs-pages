Create an app entry.

An app entry is anything that the app can define a type for that matches the entry defs and that can be serialized to SerializedBytes.

Accepts any input that implements TryInto. The default impls from the #[hdk_entry( .. )] and entry_def!() macros include this.

e.g.

ⓘThis example is not tested
#[hdk_entry(id = "foo")]
pub struct Foo(u32);
create_entry(Foo(50))?;
